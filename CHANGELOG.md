# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- **Gemini 2.5 Flash Live Integration** - Unlimited RPM/RPD for AI price extraction
- **Google Cloud Function** - Real-time broadband price scraping
- **WebSocket Support** - Bidirectional streaming for live AI models
- **GitLab Runner Setup** - Windows runner for CI/CD pipelines
- **Comprehensive Documentation** - Health reports, deployment guides, API setup
- File storage page with 1GB free storage per user
- Login and register modals on file storage page
- Comprehensive code review and improvements
- ESLint configuration for code quality
- Security improvements (XSS prevention, input validation)

### Changed
- **Broadband Checker** - Now uses AI-powered real-time price extraction
- **API Configuration** - Updated to use Gemini Live models by default
- **Deployment Scripts** - Updated with correct API keys and configuration

### Fixed
- **Linter Warnings** - Fixed import warnings in Cloud Functions
- **API Key Configuration** - Corrected GitLab CI/CD variables
- **Runner Configuration** - Fixed PowerShell executor and parallel jobs
- File storage authentication detection
- Login button functionality on file storage page
- Memory leaks in loader.js and animations.js
- Event listener cleanup across multiple components

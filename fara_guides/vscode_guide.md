# VS Code Cheat Sheet

## Common Shortcuts
- **Command Palette**: `Ctrl+Shift+P` (Use this to run any command)
- **Quick Open**: `Ctrl+P` (Find files by name)
- **Toggle Terminal**: `` Ctrl+` ``
- **Split Editor**: `Ctrl+\`
- **Format Document**: `Shift+Alt+F`

## Terminal Integration
- To open a new terminal: `Ctrl+Shift+`
- To kill terminal: Click trash icon or `exit` command.

## File Management
- **Save**: `Ctrl+S`
- **Save All**: `Ctrl+K S`
- **Close Tab**: `Ctrl+W`

## Search
- **Find in File**: `Ctrl+F`
- **Replace in File**: `Ctrl+H`
- **Find in Project**: `Ctrl+Shift+F`
